# Variable: COMMENT\_CALLDATA\_SUFFIX\_DELIMITER

> `const` **COMMENT\_CALLDATA\_SUFFIX\_DELIMITER**: `"0xe090cb6a"`

Defined in: [packages/sdk/src/constants.ts:34](https://github.com/ecp-eth/comments-monorepo/blob/main/packages/sdk/src/constants.ts#L34)

The delimiter for the comment call data suffix used to identify comment data appended to the end of a transaction.
This is the keccak256 hash of the string "ethcomment" sliced to the first 4 bytes.
