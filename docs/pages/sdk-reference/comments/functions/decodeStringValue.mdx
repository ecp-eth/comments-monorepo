# Function: decodeStringValue()

```ts
function decodeStringValue(encodedValue): Json;
```

Defined in: [packages/sdk/src/comments/metadata.ts:176](https://github.com/ecp-eth/comments-monorepo/blob/main/packages/sdk/src/comments/metadata.ts#L176)

Decodes a string value from encoded metadata bytes.

If the string is valid JSON value, it returns the parsed JSON value. Otherwise it returns string.

## Parameters

### encodedValue

`` `0x${string}` ``

The hex-encoded bytes

## Returns

[`Json`](/sdk-reference/comments/type-aliases/Json.mdx)

The decoded string value. If the value can't be decoded, it will return an empty string.
