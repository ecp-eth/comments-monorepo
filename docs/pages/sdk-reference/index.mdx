# SDK Reference

The SDK provides several modules to help you integrate the comments system into your application:

## Core Module

The core module provides the fundamental utilities for interacting with the protocol:

```typescript
import { 
  createCommentData,
  createCommentSignTypedDataArgs,
  fetchComments 
} from '@ecp.eth/sdk'
```

[View Core Module Documentation →](/sdk-reference/default/index.mdx)

## React Module

The React module provides React-specific components and hooks for easy integration:

```typescript
import { 
  CommentsEmbed,
  useGaslessTransaction 
} from '@ecp.eth/sdk/react'
```

[View React Module Documentation →](/sdk-reference/react/index.mdx)
