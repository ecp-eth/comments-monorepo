## Usage Guidelines

- Use the dark mode logo (`logo-dark.svg`) on light backgrounds
- Use the main logo (`logo.svg`) on dark backgrounds
- Maintain the aspect ratio when resizing
- Provide adequate padding around the logo
- Do not modify the colors or alter the logo design
