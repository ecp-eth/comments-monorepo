# Next.js Boilerplate App

[Demo](https://demo.ethcomments.xyz/) | [Source code](https://github.com/ecp-eth/comments-monorepo/tree/main/apps/demo)

This is a [Next.js](https://nextjs.org) project that demonstrates core functionality of the Ethereum Comments Protocol:

- Custom commenting UI (React)
- Core protocol implementation
- Comment creation and retrieval
- Wallet integration
- App signer
- Gasless transactions
- Interaction with indexer API endpoints
