# Quick Start

Get started with the Ethereum Comments Protocol in minutes.

## Installation

:::code-group

```bash [npm]
npm install @ecp.eth/sdk
# install peer dependencies
npm install viem wagmi @tanstack/react-query @tanstack/query-core
```

```bash [yarn]
yarn add @ecp.eth/sdk
# install peer dependencies
yarn add viem wagmi @tanstack/react-query @tanstack/query-core
```

```bash [pnpm]
pnpm add @ecp.eth/sdk
# install peer dependencies
pnpm add viem wagmi @tanstack/react-query @tanstack/query-core
```

:::

## Basic Usage

::::steps

### Install dependencies

```bash
npm install @ecp.eth/sdk wagmi
```

### Render the comment box

Render `<CommentsEmbed />` component with a unique url for current page.

```tsx
import { CommentsEmbed } from "@ecp.eth/sdk/embed";

const baseOrigin = "https://blog.example.com";

export function Comments() {
  return (
    <div>
      <h3>Comments</h3>
      <CommentsEmbed
        // the uri for commenting
        uri={new URL(window.location.pathname, baseOrigin)}
      />
    </div>
  );
}
```

For more details, see [integration options: react component library](/integration-options/react-component-library) and [`<CommentsEmbed />`](/sdk-reference/embed/functions/CommentsEmbed).
